FEATURES

Document Resident Interactions

The Resident Interactions tab allows you to keep track of interactions right as they happen and easily reference back to them when needed.

Manage Important Contacts

The contacts tab allows you to compile a list of important resources so they can access them as quickly as possible when needed.

Document Incident and Duty Notes

Incident and duty round notes can be quickly documented when situations arise, making it easy to reference back later for writing reports.

Export Notes and Interactions

Resident interactions, incident notes, or duty round notes can be easily exported to apps like email, Google Docs, Microsoft Teams, and more.
